Нил Стивенсон. Криптономикон
----------------------------

Версия, адаптированная под kindle. Из-за того, что в книге много формул и рисунков, пришлось 
верстать в XeLaTeX и компилировать в PDF файл. Размеры бумаги: `9cm x 12cm`, поля равны `0.3cm`. 
Если кому-нибудь нужно собрать под другой размер — не проблема, просто обновите размеры в 
преамбуле, которые сейчас задаются такой строкой:

    \usepackage[paperwidth=9cm,paperheight=12cm,top=0.3cm,left=0.3cm,right=0.3cm,bottom=0.3cm]{geometry}
